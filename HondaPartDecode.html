Honda Part # Decode

The following is purported to be how Honda Part #'s are constructed. When cross referencing part #'s, as always, buyer beware.

Part number format:

XX

XXX

-

XXX

-

XXX

XX

function number

component number

-

product number

-

classification number

special number

For example: 22401-MT6-600 (clutch spring for a CBR600FK-L).

General part numbers are divided into 3 main sections:

  1. The first 5 digits are function number and component number. Together they define the function of the part, so 22401 in the example above defines the part as a clutch spring.

  2. The next 3 digits is the Honda product number, this defines the motorcycle model the part is for. MT6 in the example above defines that part is for a CBR600FK or L. Use the product number cross reference to look up the product code for your motorcycle.

  3. The final set of digits are the classification number and the special number. The special number is not always used. In the example above 600 is the classification number and the special number is not used. These numbers further define the part within the Honda system, for example who the supplier is.
